# Information-Retrieval-System-Reuters-21578-
Developed IR pipeline on Reuters-21578 using inverted index, TF-IDF/BM25, SBERT, &amp; Cross-Encoder models. Trained XGBoostRanker with multimodal features (BM25, TF-IDF, SBERT) to boost MAP and nDCG. Built 60 queries and interactive visualizations to demonstrate measurable transformer-based IR improvements
